context/ → Es para compartir datos globales entre componentes sin estar pasando props de padre a hijo a nieto... Por ejemplo el usuario logueado, el tema (dark/light), el idioma. Se usa con createContext de React.


Ejemplo:

// context/UserContext.jsx
// Con este contexto cualquier componente de la app puede saber
// quién es el usuario logueado sin necesidad de pasar props

import { createContext, useState } from 'react'

export const UserContext = createContext()

export function UserProvider({ children }) {
  const [user, setUser] = useState(null)

  return (
    <UserContext.Provider value={{ user, setUser }}>
      {children}
    </UserContext.Provider>
  )
}
